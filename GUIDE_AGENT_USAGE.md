## 核心改进

### 1. 格式约束
- **教育背景-课程**：必须用列表格式，禁止段落描述
- 示例：`机器学习、深度学习、模式识别`

### 2. ROI止损
- 次要板块（课程、兴趣爱好）：最多2轮对话
- 用户说"没有"时快速止损，不编造内容

### 3. 智能分级
- **核心板块**（项目、工作经历）：深入挖掘
- **次要板块**（课程、爱好）：简洁真实

```
Agent: "您学过NLP课程吗？"

```

### 优化后
```
Agent: "您学过AI相关课程吗？"
用户: "没有，我学的是传统ML"
Agent: "了解。我给您一个简洁版本：
机器学习、深度学习、模式识别
确认后我们优化更重要的项目经历。"
```

## 注意事项

1. **保持真实**：绝不编造用户没有的经历或课程
2. **格式规范**：严格遵守不同板块的格式要求
3. **ROI导向**：专注高价值点，果断放弃低价值点
